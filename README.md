# Recalibrate

Landing page for Recalibrate — a next-generation wellness resort set in nature.

## Stack

Static site. No build step. Just open `index.html` or deploy.

- HTML + Tailwind (CDN) + vanilla JS
- Google Fonts: Fraunces (display) + Inter (body)
- Unsplash imagery for placeholders

## Deploy to Vercel

```bash
vercel
```

Or push to GitHub and import the repo in the Vercel dashboard — it auto-detects as a static site and deploys instantly.

## Local preview

Open `index.html` directly, or:

```bash
npx serve .
```
